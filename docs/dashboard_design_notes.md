# Dashboard Design Notes

This document describes the color logic used in the Revenue Metrics Dashboard.

## Color Logic

The dashboard uses a limited color palette to improve readability and keep the visual design consistent.

- Blue: core revenue metrics, such as MRR and LTV;
- Teal: user-based metrics, such as Paid Users, New Paid Users, and LT;
- Green: positive revenue growth factors, such as New MRR, Expansion MRR, and Back from Churn MRR;
- Orange: churn-related revenue loss;
- Red: contraction-related revenue loss;
- Dark gray: overall Revenue Change line.

## Purpose

The goal of this color logic is to make the dashboard easier to read and help users quickly distinguish between revenue growth, revenue loss, user metrics, and overall revenue change.

## Next Step

Apply this color logic consistently across all Tableau worksheets and the final dashboard layout.
